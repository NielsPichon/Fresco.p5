Class: Particle

Fresco.Particle(position)

Class for handling particles. Particles are Points which also have a velocity, a mass, a life time, and possibly varying color and transform over life. Particles may be handled in a cinematic fashion or with a physics system

Constructor

new Particle(position)

Parameters:
Name Type Description
position p5.Vector Position of the particle when spawned
Properties:
Name Type Default Description
lifetime number -1 Lifetime of the particle. If negative, the particle will live forever.
mass number 1 Mass of the particle, used when simulating physics
leaveTrail boolean false Whether the particle should leave a trail. If true, the trail will contain a shape in which a new vertex will be added with this particle current properties, each time the particle is updated. This will allow for drawing the whole trajectory of the particle at once, or simply the displacement since the previous position.
maxTrailLength number -1 Maximum length of the the trail (in number of successive positions)
colorOverLife Array.<Array.<number>> Colors to interpolate between during the life of the particles. Each color should be an array of 4 RGBA values in the range [0, 255].
colorOverLifeTime Array.<number> Timestamps at which the particle should reach a given color. This array should have the same length as the colorOverLife array.
colorInterpolation boolean linear Color interpolation mode, either `linear` or `constant`
scaleOverLife Array.<p5.Vector> Scales to interpolate between over the life of the particle.
scaleOverLifeTime Array.<number> Timestamps at which the particle should reach a given scale. This array should have the same length as the colorOverLife array.
scaleInterpolation boolean linear Scale interpolation mode, either `linear` or `constant`
rotationOverLife Array.<number> Rotations to interpolate between over the life of the particle.
rotationOverLifeTime Array.<number> Timestamps at which the particle should reach a given rotation. This array should have the same length as the colorOverLife array.
rotationInterpolation boolean linear Rotation interpolation mode, either `linear` or `constant`
simulatePhysics boolean false Whether to simulate physics or use a purely cinematic approach for handling the particle motion.
handleCollisions boolean false Whether to simulate collisions. WARNING: Currently not supported
lastUpdate number Time of last update.
birthDate number Time at creation of the particle
acceleration p5.Vector 0,0 Current acceleration of the particles. Used only if `simulatePhysics==true`
velocity p5.Vector 0,0 Current velocity of the particle
isDead boolean false Whether the particle is dead. If it is it cannot be drawn, will not be updated and will be removed at the end of the next simulation step. Particles should not be manually deleted and this boolean should be used instead.
stopSimulate boolean false Prevents the particle from further being updated. The particle will remain alive and can still be drawn.
previousPosition Fresco.Point null Position at the previous timestep.
normalizeV boolean false If True, velocity will be normalized before applying to position
